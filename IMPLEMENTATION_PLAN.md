# FamilyWall 开发计划

## 阶段：Phase 1 - 基础完善

## 目标
完善照片管理、评论优化、用户体验，打造稳定的 MVP

## 任务清单

### Sprint 1: 照片管理增强
- [ ] 照片批量选择功能
- [ ] 照片批量删除
- [ ] 照片排序（时间正序/倒序、名称）
- [ ] 相册内照片筛选

### Sprint 2: 评论与互动优化
- [ ] 评论头像显示（根据用户名生成）
- [ ] 评论时间格式优化（相对时间）
- [ ] 评论删除功能

### Sprint 3: 用户体验细节
- [ ] 空状态设计优化
- [ ] 加载状态骨架屏
- [ ] 网络断开提示
- [ ] 操作成功/失败 toast 提示
- [ ] 移动端底部导航栏

### Sprint 4: 响应式与无障碍
- [ ] 平板布局优化
- [ ] 键盘导航支持
- [ ] 焦点管理优化
- [ ] ARIA 标签完善

## 设计规范
- 风格：温馨、现代、简约
- 配色：奶油色基调 + 琥珀/橙色强调
- 字体：Noto Sans SC + Playfair Display

## 代码规范
- 遵循 clean-code 原则
- 简洁、无过度工程
- 组件单一职责
