## 1. 设计理念

### 1.1 核心关键词

| 关键词 | 解释 | 设计体现 |
|--------|------|----------|
| **温暖** | 家庭氛围，温馨感 | 暖色调、圆角、柔和阴影 |
| **回忆** | 时间感，怀旧感 | 时光轴、照片墙、老照片滤镜 |
| **连接** | 家族纽带，凝聚力 | 家族树、成员头像聚合 |
| **简洁** | 易用，清晰 | 大量留白、清晰层级 |

## 8. 动画效果

### 8.1 动画时长

```
极快: 100ms - 颜色变化
快速: 200ms - 按钮悬浮、输入框聚焦
正常: 300ms - 卡片悬浮、弹窗出现
慢速: 500ms - 页面切换
```

### 8.2 动画关键帧

```css
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}
@keyframes slideUp {
  from { opacity: 0; transform: translateY(20px); }
  to { opacity: 1; transform: translateY(0); }
}
@keyframes scaleIn {
  from { opacity: 0; transform: scale(0.9); }
  to { opacity: 1; transform: scale(1); }
}
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}
@keyframes shimmer {
  0% { background-position: -200% 0; }
  100% { background-position: 200% 0; }
}
```
